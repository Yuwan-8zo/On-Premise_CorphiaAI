# NVIDIA DLI Agent Development Skills

這個資料夾包含基於 **NVIDIA Deep Learning Institute** 課程的 Agent 開發技能集。

## 📚 可用技能

### 1. **Agentic Chatbot Builder** (初學者)
路徑: `agentic_chatbot_builder/SKILL.md`

學習建構有狀態的對話系統，包含：
- 系統訊息設計
- 多輪對話管理
- 記憶與上下文處理
- 錯誤處理機制

---

### 2. **Structured Output Generator** (中級)
路徑: `structured_output_generator/SKILL.md`

掌握結構化輸出技術：
- Pydantic Schema 定義
- Guided Generation
- 複雜巢狀結構處理
- JSON 驗證與解析

---

### 3. **Tool Calling Integration** (中級)
路徑: `tool_calling_integration/SKILL.md`

整合外部工具與函式呼叫：
- 工具定義與註冊
- 客戶端/伺服器端選擇
- ReAct 迴圈整合
- 常用工具範例

---

### 4. **RAG System Builder** (進階)
路徑: `rag_system_builder/SKILL.md`

建構檢索增強生成系統：
- 文件分塊策略
- 向量搜尋與索引
- 重排序整合
- 查詢優化技術

---

### 5. **LangGraph Workflow Builder** (進階)
路徑: `langgraph_workflow_builder/SKILL.md`

使用 LangGraph 建構複雜工作流：
- 狀態圖設計
- 節點與邊定義
- 條件路由
- 記憶與中斷機制

---

### 6. **Multi-Agent Orchestration** (進階)
路徑: `multi_agent_orchestration/SKILL.md`

協調多個 Agent 協同工作：
- CrewAI 框架使用
- LangGraph 多 Agent 模式
- 角色定義與任務分配
- 階層式協作架構

---

## 🛠️ 技術棧

所有技能基於以下技術：

- **Python 3.8+**
- **LangChain / LangGraph**
- **Pydantic 2.0+**
- **NVIDIA NIMs** (LLM 推理)
- **FAISS** (向量搜尋)
